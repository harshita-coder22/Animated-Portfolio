# 🌐 Animated Portfolio Website

A modern, responsive, and visually engaging **Frontend Developer Portfolio** built using **HTML, CSS, and Tailwind CSS**. This project showcases creative UI design, smooth animations, and a professional layout suitable for personal branding.

---

## 🚀 Live Demo

🔗 https://animated-portfolio-theta-teal.vercel.app/

> Replace the link above with your actual Vercel URL.

---

## ✨ Features

* 🎨 Modern UI/UX design with glassmorphism effects
* ⚡ Smooth animations and transitions
* 📱 Fully responsive (mobile, tablet, desktop)
* 🌈 Gradient text and glowing effects
* 🧩 Sections included:

  * Hero Section
  * About Me
  * Skills
  * Projects
  * Contact Form
* 🎯 Clean and structured code

---

## 🛠️ Tech Stack

* **Tailwind CSS (CDN)**

This project is deployed using **Vercel**.

To deploy your own version:

1. Push your code to GitHub
2. Go to Vercel
3. Import your repository
4. Deploy (no build settings required)
